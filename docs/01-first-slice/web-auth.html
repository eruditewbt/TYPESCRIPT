Web Auth (React + TypeScript)

This is the frontend of your system. It handles login, registration, token storage, and loading the authenticated user.

Core Responsibilities

  • Collect email + password
  • Send requests to API
  • Store authentication token
  • Fetch current user
  • Render dashboard

Auth Flow


User submits form
  ↓
POST /auth/register OR /auth/login
  ↓
Receive token
  ↓
Save to localStorage
  ↓
Call GET /me
  ↓
Render dashboard

Key State


const [email, setEmail]
const [password, setPassword]
const [token, setToken]
const [user, setUser]

These control the entire authentication system on the frontend.

API Call Example


fetch("http://localhost:3000/auth/login", {
  method: "POST",
  headers: {
    "Content-Type": "application/json"
  },
  body: JSON.stringify({ email, password })
})

Token Storage


localStorage.setItem("ts_academy_token", token);

This allows the user to stay logged in across refreshes.

Fetching Current User


fetch("/me", {
  headers: {
    Authorization: `Bearer ${token}`
  }
})

UI States

  • Unauthenticated → show form
  • Loading → show progress
  • Authenticated → show dashboard
