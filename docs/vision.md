# Vision

TypeScript Academy is a product-first education company where learning happens through contribution to real software.

## Strategic thesis

- Students learn faster when they build software other people actually use.
- Collaboration is a teachable skill and should be part of the curriculum from the start.
- The platform should become more valuable as students contribute new code, content, templates, and products.
- Monetization should begin with the first useful product, not only after the full ecosystem is complete.

## Core loop

1. Teach a concept.
2. Apply it to a live product.
3. Review and deploy the work.
4. Turn the resulting asset into reusable platform value.
5. Give students a path to earn from what they build.

## Anchor product

The first anchor product is the TypeScript SaaS Starter Platform:

- CLI for project generation
- Backend platform for auth, billing, and organization management
- Web dashboard for operators and customers

Everything else extends this product rather than creating disconnected course projects.
