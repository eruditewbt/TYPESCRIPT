# Product Ladder

Each stage is both a curriculum layer and a product layer.

## Stage 0: Foundations

- Learning focus: TypeScript basics, Node.js basics, Git, debugging
- Product contribution: shared types, small CLI utilities, starter bug fixes
- Monetization link: prepares contributors for the paid product ecosystem

## Stage 1: Dev Tools

- Learning focus: CLI design, file system, configuration, testing
- Product contribution: course CLI, scaffolding templates, environment helpers
- Monetization link: premium templates and paid starter kits

## Stage 2: Type Engine

- Learning focus: advanced types, generics, schemas, validation, code generation
- Product contribution: shared contracts, SDK generation, validation tooling
- Monetization link: API tooling SaaS and contract automation tools

## Stage 3: Backend Core

- Learning focus: APIs, auth, databases, billing, roles, background jobs
- Product contribution: reusable backend SaaS engine
- Monetization link: boilerplate sales, hosted backend features, subscriptions

## Stage 4: Web App

- Learning focus: React, component systems, API integration, testing, accessibility
- Product contribution: control panel and contributor dashboard
- Monetization link: dashboard templates, UI kits, white-label panels

## Stage 5: Mobile App

- Learning focus: React Native, offline sync, notifications, mobile UX
- Product contribution: mobile companion for progress and operations
- Monetization link: mobile premium access and enterprise monitoring tools

## Stage 6: Monorepo and Shared Infrastructure

- Learning focus: package architecture, build systems, shared contracts, release flows
- Product contribution: scalable developer platform
- Monetization link: enterprise starter repos and consulting

## Stage 7: Collaboration Platform

- Learning focus: team workflows, reviews, contributor systems, reputation design
- Product contribution: dev hub, team matching, contribution tracking
- Monetization link: paid workspaces, hiring marketplace, mentorship programs

## Stage 8: Deployment and DevOps

- Learning focus: CI/CD, Docker, environments, observability, release management
- Product contribution: one-click deployment and operational tooling
- Monetization link: hosting, build minutes, managed deployment subscriptions

## Stage 9: Marketplace and Monetization

- Learning focus: productization, pricing, payments, creator operations
- Product contribution: marketplace for templates, plugins, and SaaS assets
- Monetization link: transaction fees, featured listings, launch packages
